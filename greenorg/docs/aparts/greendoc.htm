h2 Misc. rules:

1. A web page in greenstand.org/docs is named *.php. It begins and ends with blocks of PHP code. For details, see /docs/aparts/template.php

2. greenstand.org/*.php does not understand relative links.
All links must be relative to the root, or absolute:
/docs/* or https://greenstand.org/docs
